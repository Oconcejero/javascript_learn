RETURN

Navegar por el DOM por elementos

(ARRAY) children

Devuelve una lista de elementos HTML hijos

(ELEMENT) parentElement

Devuelve el padre del elemento o NULL si no tiene

(ELEMENT) firstElementChild

Devuelve el primer elemento hijo

(ELEMENT) lastElementChild

Devuelve el último elemento hijo

(ELEMENT) previousElementSibling

Devuelve el elemento hermano anterior o NULL si no tiene

(ELEMENT) nextElementSibling

Devuelve el elemento hermano siguiente o NULL si no tiene

    //Documento con clases e IDs para usar selectores
    <!DOCTYPE html>
  <html lang="es">
  <body>
    <div id="app">
      <div class="header">
        <h1>Titular</h1>
      </div>
      <p>Párrafo de descripción</p>
      <a href="/">Enlace</a>
    </div>
  </body>
  </html>

USO


  document.body.children.length; //1 - Solo tiene un hijo
  document.body.children; //<div id="app"> - Solo un hijo que es "app"
  document.body.parentElement; //<html> - El padre de "body" que es HTML

  const app = document.querySelector("#app"); //Selecciona "#app"

  app.children; //[div.header, p, a] - Hijos de "#app"
  app.firstElementChild; //<div class="header">
  app.lastElementChild; //<a href="/">

  const a = app.querySelector("a");

  a.previousElementSibling; //<p>
  a.nextElementSibling; //null - No tiene porque tienen que ser hermanos

Navegar por el DOM por nodos

(ARRAY) childNodes

Devuelve una lista de nodos hijos. Incluyendo nodos de texto y comentarios

(NODE) parentNode

Devuelve el nodo padre del nodo o NULL si no tiene

(NODE) firstChild

Devuelve el primer nodo hijo

(NODE) lastChild

Devuelve el último nodo hijo

(NODE) previousSibling

Devuelve el nodo hermano anterior o NULL si no tiene

(NODE) nextSibling

Devuelve el nodo hermano siguiente o NULL si no tiene

    //Documento con clases e IDs para usar selectores
    <!DOCTYPE html>
  <html lang="es">
  <body>
    <div id="app">
      <div class="header">
        <h1>Titular</h1>
      </div>
      <p>Párrafo de descripción</p>
      <a href="/">Enlace</a>
    </div>
  </body>
  </html>

USO


  document.body.childNodes.length; //3 - Serían el "div", el "p" y el "a"
  document.body.childNodes; //[text(salto de línea), div#app, text(salto de línea)]
  document.body.parentNode; //<html> - El padre de "body" que es HTML

  const app = document.querySelector("#app"); //Selecciona "#app"

  app.childNodes; //[text(salto de línea), div.header, text, p, text, a, text] - Hijos de "#app"
  app.firstChild.textContent; //[text(salto de línea), div.header, text, p, text, a, text]
  app.lastChild.textContent; //[text(salto de línea), div.header, text, p, text, a, text]

  const a = app.querySelector("a");

  a.previousSibling; //#text
  a.nextSibling; //#text
