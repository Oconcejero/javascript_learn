EVENTOS

RETURN

Los eventos son el idioma en el que Javascript entiende las acciones del usuario.

Métodos

Mediante HTML

Mediante Javascript

Mediante addEventListener

Método aconsejado

Mediante HTML


   <script>
      funtion mostrarAlert() {
        alert('Botón pulsado');
      }
   </script>

   <button oneclick="mostrarAlert()">Prueba de botón</button>

Mediante Javascript


      <button>Prueba de botón</button>
  
        <script>
        const button = document.querySelector("button");
        button.oneclick = function() {
          alert("hello!")
        }
      </script>

Mediante addEventListener

Se puede realizar sobre cualquier elemento del DOM

Permite añadir una escucha del evento indicado (primer parámetro) y en el caso de que ocurra, se ejecutará la función asociada indicada (segundo parámetro)


      const button = document.querySelector("button");
      button.addEventListener("click", function() {
        alert("Hello!");
      });

      //Por medio de función
      
      function mostrarConsole(){
        console.log('Escribiendo algo por consola');
      }
      
      const button = document.querySelector("button");
      button.addEventListener("click", mostrarConsole);

      // Se puede eliminar "Escuchadores" con la función .removeEventListener

Objeto Event

Cuando se produce un evento se produce una instancia del objeto event


      const button = document.querySelector("button");
      button.addEventListener("click", function(event) {
        alert("Hello!");
        console.log(event);
      });

Propiedades

altKey

Devuelve si la tecla [Alt] fue pulsada durante el evento

shiftKey

Devuelve si la tecla [Mayus] fue pulsada durante el evento

ctrlKey

Devuelve si la tecla [Ctrl] fue pulsada durante el evento

button

Devuelve el botón del ratón que activó el evento

charCode

Contiene el valor Unicode de la tecla que se pulsó

clientX

Coordenada X del ratón con respecto a la ventana

clientY

Coordenada Y del ratón con respecto a la ventana

pageX

Coordenada X del evento relativa al documento completo

pageY

Coordenada Y del evento relativa al documento completo

screenX

Coordenada X del evento con respecto a la pantalla

screenY

Coordenada Y del evento con respecto a la pantalla

target

Referencia al elemento que lanzó el evento

timeStamp

Devuelve el momento en el que se creó el evento

type

Nombre del evento
