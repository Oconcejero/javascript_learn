DOM (Document Object Model) es la estructura jerárquica para representar la página web

Nodo

Es cualquier etiqueta del cuerpo, como un párrafo, el mismo body o, incluso, etiquetas de una lista

Document

Es el nodo raís, a partir del cual derivan el resto de nodos.

Element

Nodos definidos por etiquetas HTML.

Text

El texto dentro de un nodo element se considera un nuevo nodo hijo de tipo text (texto).

Attribute

Los atributos de las etiquetas definen nodos. En JAVASCRIPT no los veremos como nodos, sino como información asociada al nodo de tipo element.

Comentarios y otros

Los comentarios y otros elementos como las declaraciones doctype en cabecera de los documentos HTML generan nodos.
