Clases, classList y Métodos de classList

Modificar la clase de los objetos

.classList

Devuelve la lista de clases del elemento HTML

.classList.length

Devuelve el número de clases del elemento HTML

.classList.item(n)

Devuelve la clase número "n" del elemento HTML
NULL si no existe

.classList.contains(clase)

Indica si la clase existe en el elemento HTML (true/false)

.classList.add(" ")

Añade una clase

.classList.remove(" ")

Remueve una clase

.classList.replace(" ")

Reemplaza una clase por otra

.classList.toggle(" ")

Si no tiene la clase especificada la agrega. Si ya la tiene la elimina

    //Documento con clases e IDs para usar selectores
    <!DOCTYPE html>
  <html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Primeras pruebas con selectores</title>
  </head>
  <body>
    <h1 class="titulo">Elemento a modificar</h1>
    <script src="codigo.js"></script>
  </body>
  </html>

USO


    const titulo = document.querySelector(".titulo");

    titulo.classList.add("grande") //Añade grande a la clase
    titulo.classList.remove("grande") //Elimina grande de la clase
    titulo.classList.item(1) //Indica el valor que haya en la clase
    titulo.classList.contains("grande") //Devuelve true o false si está en la clase
    titulo.classList.replace("grande","chico") //Reemplaza una clase por otra
    titulo.classList.toogle("grande") //Si la clase está la elimina y si no está la agrega
    titulo.classList.toogle("grande",true) //Se puede forzar para que siempre la agregue y la deje

Obtención y modificación de elementos

textContent

Devuelve el texto de cualquier nodo

innerHTML

Devuelve en contenido HTML de un elemento

outerHTML

Devuelve el código HTML completo de un elemento incluyendo etiquetas

    //Documento con clases e IDs para usar selectores
    <!DOCTYPE html>
  <html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Primeras pruebas con selectores</title>
  </head>
  <body>
    <h1 class="titulo">Elemento a <b style="visibility: hidden;">modificar</b></h1>
    <script src="codigo.js"></script>
  </body>
  </html>

USO


    const titulo = document.querySelector(".titulo");

    let resultado = titulo.textContent; //Elemento a modificar
    let resultado = titulo.innerHTML; //Elemento a <b style="visibility: hidden;">Modificar </b>
    let resultado = titulo.outerHTML; //<p class="titulo">Elemento a <b style="visibility: hidden;">Modificar </b></p>
