Métodos para Definir, Obtener y Eliminar valores de atributos

setAttribute( )

Modifica el valor de un atributo

getAttribute( )

Obtiene el valor de un atributo

removeAttribute( )

Remueve el valor de un atributo

    //Documento con clases e IDs para usar selectores
    <!DOCTYPE html>
  <html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Primeras pruebas con selectores</title>
  </head>
  <body>
    <input type="range" class="rangoEdad">
    <label>Rango de Edad</label>
    <script src="codigo.js"></script>
  </body>
  </html>

Modificar el atributo


    const rangoEdad = document.querySelector(".randoEdad");
    rangoEdad.setAttribute("type","text")
    // Primer valor el que está, segundo valor al que se quiere modificar

Obtener atributo


    const valorDelAtributo = rangoEdad.getAttribute("type");
    console.log(valorDelAtributo); //range
    // Indica el valor del atributo específico

Obtener atributo


    rangoEdad.removeAttribute("type");
    // Elimina el atributo. ELIMINADO, no vacío

Atributos globales

class

Lista de clases del elemento separadas por espacios

contenteditable

Indica si el elemento puede ser modificable por el usuario (bool)

dir

Indica la direccionabilidad del texto

hidden

Indica si el elemento aún no es, o ya no es, relevante

id

Define un identificador único

style

Contiene declaraciones de estilo CSS para ser aplicadas al elemento

tabindex

Indica si el elemento puede obtener un focus de input

tittle

Contiene un texto con información relacionada al elemento al que pertenece

Atributos de Inputs

Se acceden directamente desde Javascript

className

Muestra la clase

value

Muestra el valor

type

Muestra el tipo

accept

Indica que archivo es aceptado

form

Permite crear un botón fuera del formulario

minlength

Indica el número mínimo de caracteres en un input

placeholder

Muestra un mensaje en el input

required

Indica si el campo es requerido o no

    //Documento con clases e IDs para usar selectores
    <!DOCTYPE html>
  <html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Primeras pruebas con selectores</title>
  </head>
  <body>
    <input type="text" class="input-normal">
    <script src="codigo.js"></script>
  </body>
  </html>

USO


    const input = document.querySelector(".input-normal");

    console.log(input.className)//Muestra la clase - input-normal
    
    console.log(input.value)//Muestra lo que hay dentro del input
    
    console.log(input.type)//Se puede modificar desde dentro (console.log(input.type="range"))
    
    input.accept//En los input tipo FILE indica que archivo va a aceptar (input.accept="image/png")
    
    input.form //Ejecuta un input fuera del form enlazándolo con id
    
    input.minLength //Indica el número mínimo de caracteres de un input
    
    input.placeholder = "Escribe tu nombre" //Muestra un mensaje en el input
    
    input.required = "required"; //Indica si es un campo requerido o no
