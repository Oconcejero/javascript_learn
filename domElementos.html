Obtención y modificación de elementos

textContent

Devuelve el texto de cualquier nodo

innerHTML

Devuelve en contenido HTML de un elemento

outerHTML

Devuelve el código HTML completo de un elemento incluyendo etiquetas

Creación de Elementos

.createElements( )

Crea un elemento

.createTextNode( )

Crea un texto

.appenChild( )

Crea un elemento hijo

.removeChild(node)

Elimina y devuelve el nodo hijo de "node"

.replaceChild(new, old)

Reemplaza el nodo hijo de "old" por "new". Devuelve "old"

.insertBefore(new, old)

Inserta el nodo "new" antes de "node" y como hijo del nodo actual

.insertBefore(new, NULL)

Inserta el nodo "new" después del último nodo hijo. Equivalente a ".appendChild()"

.createDocumentFragment( )

Devuelve el código HTML completo de un elemento incluyendo etiquetas

API de insertar elementos

.before( )

Añade el nuevo elemento justo antes.

.after( )

Añade el nuevo elemento justo después.

.prepend( )

Se añade el nuevo elemento antes del primer hijo.

.append( )

Se añade el nuevo elemento después del último hijo.

.replaceChildren( )

Elimina todos los hijos y los sustituye por el nuevo elemento.

.replaceWith( )

Se sustituye por el nuevo elemento.

.remove( )

Elimina el propio elemento.
