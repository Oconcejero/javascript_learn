Emisión de eventos

CustomEvent

Se crea con 2 parámetros. el primer va a ser un string de como vamos a definir al evento y el segundo será un objeto


      const messageEvent = new CustomEvent("message", options);

(object)detail - NULL

Objeto que contiene la información que queremos transmitir

(Boolean)bubbles - false

Indica si el evento debe burbujear en el DOM "hacia la superficie" o no

(Boolean)composed - false

Indica si la propagación puede atravesar Shadow DOM o no

(Boolean)cancelable - false

Indica si el comportamiento se puede cancelar con .preventDefault()

      //JAVASCRIPT

stopPropagation

Para que el elemento no se propague y muera en el elemento que se ha producido


      //JAVASCRIPT
      let padre = document.querySelector(".padre");
      let hijo = document.querySelector(".hijo");

      document.body.addEventListener("click", () => {
        console.log("Evento lanzado desde body");
      })

      padre.addEventListener("click", () => {
        console.log("Evento lanzado desde padre");
      })

      hijo.addEventListener("click", (event) => {
        event.stopPropagation();
        console.log("Evento lanzado desde hijo");
      })
